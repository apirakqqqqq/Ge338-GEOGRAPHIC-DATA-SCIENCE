# Lab 2

# **1. เหตุผลที่เลือกพื้นที่ + คำถามวิจัย**

พื้นที่จังหวัดเลยถูกเลือกเนื่องจากเป็นพื้นที่ภาคเหนือที่มีทั้งป่าไม้ ภูเขา และพื้นที่เกษตรกรรมผสมกัน ทำให้เหมาะสำหรับการศึกษาการเปลี่ยนแปลงของพืชพรรณในช่วงฤดูแล้ง โดยมีคำถามวิจัยในใจคือ พื้นที่ใดมีการเปลี่ยนแปลงของความเขียวของพืชมากที่สุดระหว่างต้นฤดูแล้งและปลายฤดูแล้ง และพื้นที่ใดยังคงมีความอุดมสมบูรณ์อยู่ 

# **2. เลือกดาวเทียมอะไร เพราะอะไร**

เลือกใช้ข้อมูลจาก Sentinel-2 เนื่องจากมีความละเอียดเชิงพื้นที่สูงถึง 10 เมตร ซึ่งสามารถตรวจจับรายละเอียดของพื้นที่ขนาดเล็ก เช่น แปลงเกษตรหรือพื้นที่ป่าขนาดย่อยได้ดีกว่าดาวเทียม Landsat

3. เลือกช่วงเวลาใด และทำไมไม่เลือกช่วงอื่น

เลือกช่วงต้นฤดูแล้ง (พฤศจิกายน–ธันวาคม) และปลายฤดูแล้ง (กุมภาพันธ์–เมษายน) เนื่องจากเป็นช่วงที่ความแตกต่างของความชื้นและพืชพรรณชัดเจนที่สุด และไม่เลือกช่วงฤดูฝนเพราะค่าความเขียวของพืชจะสูงใกล้เคียงกัน ทำให้เห็นการเปลี่ยนแปลงได้ไม่ชัด

# **4. เกณฑ์กรองเมฆ และผลกระทบ**

กำหนดเกณฑ์กรองเมฆที่ประมาณ 20% เพื่อให้ได้ภาพที่มีคุณภาพดีและลด noise จากเมฆ โดยหากตั้งค่าต่ำเกินไปจะทำให้จำนวนภาพลดลงและอาจไม่มีข้อมูลเพียงพอ แต่หากตั้งค่าสูงเกินไปจะทำให้ภาพมีเมฆปะปนและลดความแม่นยำของผลลัพธ

# **5. ถ้าใช้ดาวเทียมอื่นจะต่างอย่างไร**

หากเปลี่ยนไปใช้ Landsat 8 หรือ Landsat 9 ซึ่งมีความละเอียด 30 เมตร จะทำให้รายละเอียดของพื้นที่ลดลง ส่งผลให้ค่าการเปลี่ยนแปลง NDVI ดูเรียบขึ้นและไม่สามารถตรวจจับการเปลี่ยนแปลงขนาดเล็กได้ชัดเจนเมื่อเทียบกับ Sentinel-2

# **6. ข้อจำกัดของ Index และทางเลือก**

NDVI มีข้อจำกัดในพื้นที่ที่มีพืชหนาแน่นมาก เนื่องจากค่าจะอิ่มตัวและไม่สามารถแยกระดับความหนาแน่นของพืชได้ดี รวมถึงได้รับผลกระทบจากดินและความชื้น ดังนั้นสามารถใช้ EVI เพื่อลดผลกระทบจากบรรยากาศ หรือ NDMI เพื่อวิเคราะห์ความชื้น ซึ่งอาจเหมาะสมกับพื้นที่นี้มากกว่าในบางกรณี

# **7. ข้อมูลที่ยังขาดสำหรับใช้งานจริง**

หากต้องนำไปใช้จริงกับหน่วยงานรัฐ ยังขาดข้อมูลภาคสนาม (ground truth) ข้อมูลชนิดพืช ข้อมูลสภาพอากาศ เช่น ปริมาณฝน และข้อมูลการใช้ที่ดิน ซึ่งจำเป็นต่อการยืนยันและตีความผลลัพธ์ให้แม่นยำมากขึ้น

# **8. Scale มีผลอย่างไร**

การเลือก scale ที่ละเอียด เช่น 10 เมตร จะให้ความแม่นยำสูงและแสดงรายละเอียดพื้นที่ได้ดี แต่ทำให้ไฟล์มีขนาดใหญ่และใช้เวลาในการประมวลผลมาก ในขณะที่ scale ที่หยาบขึ้น เช่น 30 เมตร จะช่วยลดขนาดไฟล์และเพิ่มความเร็ว แต่ทำให้สูญเสียรายละเอียดเชิงพื้นที่

# **9. วิธีที่เลือก**

วิธีการที่ใช้ในงานนี้คือการสร้าง composite ของภาพดาวเทียมในสองช่วงเวลา จากนั้นคำนวณค่า NDVI และนำค่าที่ได้มาเปรียบเทียบเพื่อวิเคราะห์การเปลี่ยนแปลง รวมถึงใช้การแสดงผลแบบ interactive เพื่อช่วยในการวิเคราะห์เชิงพื้นที่

# **10. เหตุผลของวิธีที่เลือก**

วิธีนี้ถูกเลือกเพราะสามารถแสดงการเปลี่ยนแปลงของพืชพรรณได้อย่างชัดเจน เข้าใจง่าย และเหมาะสำหรับการวิเคราะห์ในระดับพื้นที่กว้างโดยไม่ต้องใช้ข้อมูลที่ซับซ้อนมาก

# **11. ข้อจำกัดที่พบ**

ข้อจำกัดของงานนี้คือการใช้ข้อมูลเพียงสองช่วงเวลา ทำให้ไม่สามารถวิเคราะห์แนวโน้มระยะยาวได้ อีกทั้งยังมีข้อจำกัดจากความละเอียดของข้อมูลและผลกระทบจากเมฆ รวมถึงการประมวลผลข้อมูลขนาดใหญ่ที่ใช้เวลานาน

# **12. สิ่งที่ค้นพบ**

ผลการวิเคราะห์พบว่าหลายพื้นที่มีค่า NDVI ลดลงในช่วงปลายฤดูแล้ง แสดงถึงการลดลงของความเขียวของพืชพรรณ ขณะที่บางพื้นที่ยังคงมีค่า NDVI สูง ซึ่งอาจเป็นพื้นที่ป่าหรือพื้นที่ที่มีแหล่งน้ำสนับสนุน ทำให้สามารถระบุพื้นที่ที่ได้รับผลกระทบจากความแห้งแล้งได้อย่างชัดเจน
